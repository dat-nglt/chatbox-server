Kiểm tra trạng thái redis: sudo systemctl status redis-server

Khởi động redis: sudo systemctl status redis-server

Dừng redis: sudo systemctl stop redis-server

Khởi động lại reids: sudo systemctl restart redis-server

Tải lại redis: sudo systemctl reload redis-server

Bật Redis chạy tự động khi reboot VPS: sudo systemctl enable redis-server

Tắt auto-start: sudo systemctl disable redis-server

Kiểm tra Redis đang lắng nghe port nào: sudo lsof -i :6379

Kết nối vào Redis CLI: redis-cli
